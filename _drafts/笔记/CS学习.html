变量:
     类型分为四种：char（字符，无法表示大小），int（整数，可以递进、比大小），float、double（可表示小数）。变量名必须以字母开头。常量和变量：常量在程序中不能改变值，变量可以。
常量:
     （利用字母后缀区别不同类型常量）
整型除了十进制，还可以用八进制和十六进制表示，通过前缀表示不同进制
浮点数一样
字符型还包含有转义字符，可以比大小
枚举常量enum可以自动递增赋值
字符数组初始化：如果使用""，则数组长度需考虑'\0'
在不显式初始化的情况下，外部变量和静态变量默认初始化为0，自动变量和寄存器变量的初值则没有定义
EOF值为0
尽量使用宏定义，使得程序更加清晰
类似于&或++的一元运算符遵循从右至左的结合顺序
外部变量:
     总之，其它文件只能调用一个文件中的外部变量，而不能调用该文件中函数内的外边变量；一个函数中的变量若想外部调用，则必须在前边加上extern；函数不用加extern即可被所有文件调用；声明时不能紧跟着声明对其初始化，但可以在声明后的语句里改变该外部变量的值；若在一个文件的函数内部调用其他文件内的函数，则需在调用之前声明该函数，但不需要加extern，只有变量才有可能要加extern（或者说，一个函数内部若要调用其他函数，而其他函数并未在该函数前定义，则必须声明；再或者说，调用函数前必须声明，而定义等同于声明）；
ASCII:
     注意在ASCII中，大写字母和小写字母的间隔不是26，而是32，而且小写字母的位置要高于大写字母；
合理使用全局变量和参数：这个问题应该主要考虑函数接口（指的是函数的参数类型和返回值类型）的通用性，即尽量保证设置的函数参数适用于大多数函数，尽量避免参数在函数内无用，所以函数参数应取所有配套函数的最大公约数，然后将部分函数共享的参数设置为全局变量；例子可参考E:\Cyuyan\5-14,15,16,17qsort中函数qsort和int (*comp)(void *, void *)参数以及全局变量order、aA的设置；
函数注释很重要，应当将其写在函数开头的上一行；
void *p是通用型指针，可以使用在无法确定指针类型的指针声明、定义上，总之，是为了保证函数的通用性才使用viod *p，但是，相对于指明类型的指针，通用性指针似乎没有什么劣势，那为何不都使用void *p型指针

sizeof 
     is a compile-time unary operator,sizeof object & sizeof (type name)
     #define NKEYS (sizeof keytab / sizeof(struct key))
          Another way to write this is to divide the array size by the size of a specific element:
     #define NKEYS (sizeof keytab / sizeof(keytab[0]))
          This has the advantage that it does not need to be changed if the type changes.

alignment requirements:
     Don't assume, however, that the size of a structure is the sum of the sizes of its members.Because of alignment requirements for different objects, there may be unnamed ``holes'' in a structure. Thus, for instance, if a char is one byte and an int four bytes, the structure
struct {
char c;
int i;
};
might well require eight bytes, not five. The sizeof operator returns the proper value.
